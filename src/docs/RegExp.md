# 正则
记录常用的正则和api。

## 基础语法
| 语法           | 说明                        |   语法              |    说明             |
|-------------- |----------------------------|---------------------|------------------- |
| **.**             | 匹配除换行符的单个字符     | **/\?+*[]().**    | 特殊字符，匹配时加 \    |
| **\d**            | 匹配单个数字，同[0-9]     | **{n}, {n,}, {n,m}** | 匹配 n 次,n+次,n到m次 |
| **\D**            | 非\d                    | **?, \*, +**       | 0或一个，至少一个，任意个  |
| **\w**            | 匹配字母数字下划线，用处不大 | **^, $**          | 匹配头尾                |
| **\W**            | 非\w                    | **[123]**      | 限定内容匹配             |
| **\s**            | 匹配单个空白字符，包括\n,\r,\f,\t,\v等              | **(), $1**        | 分组与匹配              |
| **\n**            | 匹配单个换行符            |

s.replace(/[^A-Za-z0-9]/g,'');
匹配非字母非数字的字符。
