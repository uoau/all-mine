# 文本
组件库所使用的字体、行高间距等文本规范


## 字体
``` css
* {
    font-family:
        'Helvetica Neue',                 /* 英文正文字体 */
        -apple-system,BlinkMacSystemFont, /* 包括苹方 */
        'Microsoft YaHei',                /* 使用雅黑 */
        'Segoe UI Emoji',                 /* 微软系 emoji 表情 */
        'Apple Color Emoji';              /* 苹果系 emoji 表情 */
}
code {
    font-family: 'Menlo';                 /* 适用于显示代码字体 */
}
```
